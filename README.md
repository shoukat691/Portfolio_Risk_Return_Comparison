# Portfolio Risk & Return Analysis (10-Year Study)

## Project Objective
The goal of this project is to perform a comparative analysis of different asset classes to understand how individual growth stocks (Apple) perform against a diversified market index (S&P 500) and a safe-haven asset (Gold).

## Methodology
- *Data Source*: Real-time financial data extracted via the yfinance library.
- *Normalization*: All assets were normalized to a base value of 100 at the starting point (2014) to provide a fair "Apple-to-Apple" comparison of investment growth.
- *Tools*: Developed using Python, with heavy reliance on Pandas for data manipulation and Matplotlib for professional-grade visualization.

## Key Performance Insights (2014 - 2024)
- *Apple (AAPL)*: Showed exponential growth (~1011%), outperforming the benchmark but with higher volatility.
- *S&P 500 (SPY)*: Provided steady, reliable returns (~211%), acting as the standard for market performance.
- *Gold (GLD)*: Acted as a hedge with lower returns (~62%) but significantly lower correlation to stock market swings.
